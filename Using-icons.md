Using icons might be confusing at first but here's a quick guide:

Go to [Material Design icons](https://materialdesignicons.com/).

Find an icon you like and right-click it.

![Step 1](https://i.imgur.com/AyR3v0z.png)

There is an option called 'View XAML'. Press it.
You will see this window:

![Step 2](https://i.imgur.com/Z07OsNH.png)

Copy the content into your XAML file. Place it where you normally would place your images.

If the height isn't fitting, adjust the options inside the canvas or viewport.
